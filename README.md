## Installation

1. **Prerequisites**:
   - Python 3.9 or later
   - MySQL Server
   - XAMPP (recommended for easy MySQL setup)

2. **Setup Database**:
   - Create a MySQL database named `library_management`
   - Run the SQL script from `database/setup_db.sql`
